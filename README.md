# 社員管理アプリ（シンプル版）

## 概要

TypeScript を使用したシンプルな社員管理システム

## 技術スタック

- バックエンド: Fastify + TypeScript
- フロントエンド: React + TypeScript
- データベース: MySQL
- インフラ: AWS (App Runner, S3, CloudFront)

## 開発環境

- Node.js
- TypeScript
- MySQL

## プロジェクト構成

- `/api` - バックエンド API
- `/web` - フロントエンドアプリケーション
